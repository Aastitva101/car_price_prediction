# Car Price Prediction
## Description:-
This project predicts car prices using machine learning based on features like mileage, engine, fuel type, etc.
## Model Used:-
Linear Regression
## Tech Stack:-
- Python
- Pandas
- Numpy
- Scikit-learn
- Matplotlib/Seaborn
## How to Run (Google Colab)
1.Open Google Colab

2.Upload the notebook file:
- car_price_prediction.ipynb

3.Upload the dataset file:
- car data.csv

4.Run all cells step-by-step
## Output
- Predicts car prices
- Shows Model Performance

## Project Files
- car_price_prediction.ipynb → main notebook
- car data.csv → dataset used for training
